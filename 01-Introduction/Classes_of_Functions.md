## Time Functions
### Types of Time Functions
| Label | Time Functions |
|---|---|
| Constant</br> _The complexity is independent of size of user input_| O(1) |
| Logrithemic | O(logn) |
| Linear | O(n) |
| Quadratic | O($`n^2`$) |
| Cubic | O($`n^3`$) |
| Exponentical | O($`2^n`$) </br> O($`3^n`$) </br> O($`n^n`$) |
